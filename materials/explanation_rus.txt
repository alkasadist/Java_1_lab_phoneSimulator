Порождающими паттернами выбраны:
    Multiton - для исключения создания телефонов с одинаковыми номерами.
    Builder - для удобства создания телефонов (через цепочку вызовов методов).

Структурными паттернами выбраны:
    Facade - для упрощения взаимодействия между объектами.
    Proxy - для отделения логики ошибок и исключений.

Поведенческими паттернами выбраны:
    Mediator - для управления звонками между телефонами через отдельный интерфейс.
